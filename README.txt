README 
In order to run go to the main MapMaker folder on your terminal. 
There will be two sample .txt files and two executable .jar files
If you want to run local search type: java -jar MapMaker_LS.jar
Likewise, if you want to run back tracking search type: java -jar MapMaker_BT.jar
On both files once started it will prompt you to type the file name and path of the .txt file that you want to go over.
It will keep asking you until you type the path of a readable .txt file.
After that it will run the search show the pairs of states and colors and the nodes it had to traverse.
